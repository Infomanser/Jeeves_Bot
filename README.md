# Jeeves Bot

Кишеньковий асистент на Python + Termux.

## Функції
- 📅 Календар подій
- 🌤 Погода (Open-Meteo)
- 🗞 Стрічка новин (RSS)
- 🔦 Керування ліхтариком та TTS (Termux API)
- 📊 Моніторинг системи

## Встановлення на Termux
1. `pkg install python termux-api git`
2. `git clone ...`
3. `pip install -r requirements.txt`
4. Створити `.env` (див. config.py)
